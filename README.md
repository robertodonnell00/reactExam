# React Movies App

A simple React web application that displays movies using data from The Movie Database (TMDB) API.  
Users can browse movies, add them to their favourites or playlist, and view detailed information about each film.
Assignment for Web App Dev 2. 2025.

## Features
- Browse popular, top-rated, trending, and now-playing movies  
- View detailed movie information and credits  
- Add and remove favourites  
- Add and remove movies from a playlist  
- Write and view reviews  
- Filtering options  
- Dark themed UI using Material-UI (MUI)

## Technologies Used
- React  
- React Router  
- React Query  
- Material-UI (MUI)  
- TMDB API  
- JavaScript

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/your-repo-name.git
   ```
2. Navigate to the project folder:
   ```bash
   cd your-repo-name
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Running the Application
1. Create a `.env` file in the project root and add your TMDB API key:
   ```
   VITE_TMDB_KEY=your_api_key_here
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open your browser and visit:
   ```
   http://localhost:5173
   ```
